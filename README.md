Developing a predictive model for radiosensitivity in cervical cancer radiotherapy using CT radiomic and clinical features.

### 📂 Project Structure / Codebase

- `extract_features.py`: Extracts radiomic and clinical features.
- `model_pipeline.py`: Handles feature selection, model training, and evaluation.
- `shap_analysis_rf.py`: Performs explainability analysis for the Random Forest model.
- `R/`: Contains R scripts for generating correlation plots.
